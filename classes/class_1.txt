Kitchen

Attributes:
hasOven (boolean)
utensils (array)
numOfPlates (integer)
headChef (string)

Methods:
hireChef (changes value of headChef)
servePlate (accepts number of customers dining at a table as
  an argument and subtracts that from numOfPlates)
bakeMeal (conditional; accepts a menu item in the form of a
string as an argument requires hasOven === true before proceeding)
chooseUtensil (selects utensil based on called element index position)
